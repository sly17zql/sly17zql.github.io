### 使两个整数相等的位更改次数

给你两个正整数 n 和 k。

你可以选择 n 的 二进制表示 中任意一个值为 1 的位，并将其改为 0。

返回使得 n 等于 k 所需要的更改次数。如果无法实现，返回 -1。

### 解答

```python
class Solution:
    def minChanges(self, n: int, k: int) -> int:
        return -1 if n & k != k else (n ^ k).bit_count()
```

### 思路

无法实现返回-1，的情况，即 k 不是 n的子集情况。

如果k 为 n的子集，那么直接取两者的异或的1的数量就是答案
