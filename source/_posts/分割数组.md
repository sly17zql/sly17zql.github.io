### 分割数组

给你一个长度为 偶数 的整数数组 nums 。你需要将这个数组分割成 nums1 和 nums2 两部分，要求：

nums1.length == nums2.length == nums.length / 2 。
nums1 应包含 互不相同 的元素。
nums2也应包含 互不相同 的元素。
如果能够分割数组就返回 true ，否则返回 false 。

### 解答

```python
class Solution:
    def isPossibleToSplit(self, nums: List[int]) -> bool:
        c = defaultdict(int)
        for n in nums:
            c[n] += 1
            if c[n] >= 3:
                return False
        return True
```

### 思路

直接hashmap，判断是否有超过2次的。

### 复杂度

时间复杂度：O(N)

空间复杂度：O(N)
