收集所有金币可获得的最大积分

有一棵由 n 个节点组成的无向树，以 0 为根节点，节点编号从 0 到 n - 1 。给你一个长度为 n - 1 的二维 整数 数组 edges ，其中 edges[i] = [ai, bi] 表示在树上的节点 ai 和 bi 之间存在一条边。另给你一个下标从 0 开始、长度为 n 的数组 coins 和一个整数 k ，其中 coins[i] 表示节点 i 处的金币数量。

从根节点开始，你必须收集所有金币。要想收集节点上的金币，必须先收集该节点的祖先节点上的金币。

节点 i 上的金币可以用下述方法之一进行收集：

  • 收集所有金币，得到共计 coins[i] - k 点积分。如果 coins[i] - k 是负数，你将会失去 abs(coins[i] - k) 点积分。

  • 收集所有金币，得到共计 floor(coins[i] / 2) 点积分。如果采用这种方法，节点 i 子树中所有节点 j 的金币数 coins[j] 将会减少至 floor(coins[j] / 2) 。

返回收集 所有 树节点的金币之后可以获得的最大积分。

思路

因为本身是一棵树，先构建无向图，然后可以从root节点开始进行深度遍历。递归函数计算结果，f(n, f, p)，需要维护三个值当前节点，偏移量，也就是积分进行 /2 操作的次数，以及父节点。因为题中，数值最大不超过10^4，所以当偏移量超过13的时候，就可以不需要选择第二种了。

解答

复杂度

coins长度：N，coins最大值：M

时间复杂度：O(NlogM)

空间复杂度：O(NlogM)
