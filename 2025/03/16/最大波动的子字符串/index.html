最大波动的子字符串

字符串的 波动 定义为子字符串中出现次数 最多 的字符次数与出现次数 最少 的字符次数之差。

给你一个字符串 s ，它只包含小写英文字母。请你返回 s 里所有 子字符串的 最大波动 值。

子字符串 是一个字符串的一段连续字符序列。

思路

看到数据范围 10^4，想到26个字母遍历组合，10^2，时间复杂度可以符合要求。

  • 首先遍历组合a，b两种，
  • 对于确定a，b后，可以是用动态规划来计算波动率
  • 波动率的计算需要注意的是，如果b没有出现过，那么波动率是没有意义的
    • 定义宽度为2的stack，stack[i][0]表示当前最大波动值，stack[i][1]表示当前携带b的最大波动率
    • 维护stack[i+1][0]
      • 如果s[i+1] == a，则stack[i+1][0] = stack[i][0] + 1
      • 如果s[i+1] == b，则stack[i+1][0] = max(stack[i][0]-1, 0)
      • 如果s[i+1] != a 且 != b，则stack[i+1][0] = stack[i][0]
    • 维护stack[i+1][1]
      • 如果s[i+1] == a，则stack[i+1][1] = stack[i][1] + 1
      • 如果s[i+1] == b，则stack[i+1][1] = stack[i][0] - 1
      • 如果s[i+1] != a 且 != b，则stack[i+1][1] = stack[i][1]
    • 并在每次计算当前以后维护当前cur_ans

解答

复杂度

字符串长度：N

时间复杂度：O(N * M)，M为26的排列组合数

空间复杂度：O(1)
