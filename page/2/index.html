小学往事

发表于 2025-01-12 分类于 随笔

昨日看了某个关注的博主的文章后，也想到了小学时候的一些故事，来聊一聊这些事。

出生在一个小的县城，小学是在县里的镇中心小学就读。小学时，我总感觉时间过得好慢，过了好久好久，还是一个小学生，在这漫长的小学时光中，印象最深有两件事。

一是竞选班干部。 记得那时候，小组长和班干部都会带上一个袖标，上面是一档杠或者两档杠，表明自己是个班干部，如果你能够在当上学校的干部你可以佩戴三档杠。我心目中特别向往有一天也能带上杠。但是那时候的我从未在全班面前说过话，更是从没竞选过班干部。但是在对杠的渴望下，我还是想要找一个岗位(我暂时没有找到更准确的词来表达这个意思)。我思来想去，唯一觉得还算有自信的就是数学课代表，基于从小就喜欢数学，自己数学成绩在班里也算可以。当这个学期的数学课代表的竞选活动那天，记得当时有特别多的同学上了讲台，我特别紧张，每次看到有同学上去，我就会对自己说一遍：“等这个讲完我一定要上去。”直到已经上去了大约20人左右了，我却开始纠结自己还有没有希望，到底要不要上去了。最后我还是鼓足勇气上去了，然后站在讲台上，望向下面，看到黑压压的人头（那时候一班大概60多人），我好像什么话也说不出来了。我一会到自己的位置，就不记得我到底说了什么，现在回想只记得我后来就哭了，然后特别严肃地结束了后面的投票环节，全程坐得端端正正。神奇的是后来我竟然当选了数学课代表，那时候我就感觉是不是因为我哭了，别人同情我所以给我的票数特别多。当选数学课代表的第二天，我就为自己带上了那个两档杠，感觉自己特别成功，仿佛和昨天的我已经不一样了，还给家里的长辈都看了我的两条杠肩章。

另一件印象特别深刻的事就比较让人难过了，这件事我当时并没有对我印象很深，但是随着时间的推移，我对这件事情的印象愈发深。

在中国长大的学生应该都会有印象，学校很喜欢为各方面评分，例如卫生、纪律、成绩之类的，然后为各班进行排名。当时事件的起因就是，某一天，我忘记带了校牌，学校的纪律是在进学校是要检查每个人是否带校牌的，但是因为那天快迟到了，所以我也不能再回家拿了。我便径直走了进去，只希望检查的同学、老师没有看到，但是很不幸，他们火眼金睛，从涌入的一票学生中看到了我空空如也的胸口。因为我，班级被扣了0.5分，我当时特别内疚的也有点害怕。过了几天后，因为每周都会计算班级得分，可能是周五，或者第二周周一，班主任找到了我，说了什么我已经忘记了，大概就是批评我为班级抹黑之类的吧，随后打了我一个耳光。

这件事就是这么简单的一个事情，当时我都没有把这个事告诉我的父母，我也不记得是因为我无所谓，还是害怕父母知道。后来不知道这个事情，我同学是怎么知道的，他告诉了他妈妈，他妈妈告诉了我妈。这些，也是很久之后我妈才告诉我的，起初这件事对我来说就算过去了，我也没有放在心上，但是慢慢地，这件事好像对我来说越来越重要，导致我长大了也会时不时地想起这件事。可能是由于越长大越知道这件事的严重性吧，小时候觉得仅仅就是自己有错误，被老师打了而已。

说完自己的事情，我再说说我看的博主文章里描述的他的小学生活，他和我是属于同龄人，但是他小学的生活听上去，十分的压抑，随后他因为自己的一些个人遭遇，以及对中国教育的观察，批判国内的一些教育模式和管理模式。看完以后，我想到自己的小学生活，也开始思考。其中第一件事，这些条条杠杠的追逐，我认为虽然是小孩追求一个自我满足的过程，但结合当时的实际感受，确实觉得当我带上两道杠会有一种很强烈的优越感，也有点如博主所说的小官僚主义的味道。第二件事，更是一件比较严重的事情。想到当时的我，竟然还天真地觉得因为自己的过错导致了班级扣分，被老师打的是很正常的事情。但是随着人生经历的丰富，我现在很难想象，一个成年人，更别说作为老师，会用巴掌去打一个孩子一个学生，而这仅仅是因为忘带校牌而已。我也不想矫情地说，这件事对我的人生产生了什么深远的影响，但是确实让我一直记到现在。学校本该是一个教书育人，让孩子快乐成长的地方。仅仅还是10岁左右的孩童，丢三落四并不是什么不可原谅的错误，要因为这些小事扣分，被班主任打耳光，这有违学校的初衷，老师的道德。对于当时的教育理念、制度，不进行过多评价，我只希望我的孩子不要再遇到这样的事情，可以接受到好的教育，最重要的是懂得尊重每个个体，学会爱人，爱世界。

德国的教育发展

发表于 2025-01-07 分类于 胡乱看

德国作为世界的强国，经济科技军事发展一直名列前茅，得益于他先进的教育理念。世界现代教育体系的设立，很大程度上受到了德国的影响。

起初，在德国的受教权只属于贵族以及神职人员，在15世纪，马丁·路德为了推进教会的发展，提出了相关教育的一系列改革措施，并提出把教育普及到平民，这一跨时代的设想。他积极地将此付诸实际行动，发表了《论送子女入学的责任》一文，提到政府应强制所有父母将子女接受相关教育，并做好监督职责。马丁·路德的这套国家管理教育的理念促进了公共教育体系和模式的产生发展。在此基础上，德国全体人民素质得到了大幅提升，当时甚至戏称：德国的农民都比法国的贵族有学识。

到了18世纪，普鲁士王国将这一切体系化，设立了国民教育体系，包括设立了师范学院、普及全民义务教育、中小学教育改革、建立研究型大学。整体教育体系的出现，在社会挖掘人才上，难度大大降低，有越来越多有天赋的人能展现自己的才能，推动整个社会的巨大发展和进步。后来，普鲁士在战争中战败于拿破仑的法国，丧失了大量土地和财富。但是战败的背景下，普鲁士王威廉三世更是下定决心要大力发展教育，在各地开办学校，以图荟萃国家精英。虽然在百业萧条、国家破败中，发展教育举步维艰，但是普鲁士王威廉三世没有被困难所打倒，他不仅拿出了皇室所有的积蓄，最令人感动的是，普鲁士王威廉三世捐出自己的宫邸，在宫邸上成立了大学，取名为柏林洪堡大学，也就是著名的柏林大学。他曾说过一句话：“正是由于穷困才要办教育，还没听说哪个国家是办教育而亡国的，教育是民族未来的必行之路。”在他的影响下，他的儿子在位时颁布了世界上第一部义务教育法《普国普通义务教育法》。到了19世纪中叶，普鲁士王国的适龄儿童的入学率已经高达97.5%，即使放眼现在社会，也是值得称道的功绩。在先进的教育理念和体系下，普鲁士的民众遵守纪律，富于实干，文化程度远超其他国家。普鲁士也终成了欧洲的强国，打败了法国，让四分五裂的德国再次统一。

德国教育到现在也值得称道。之前提到过得柏林洪堡大学，是世界上第一所现代大学。在德国统一后，更是迎来了前所未有的巨大发展，更多的第一纷纷涌现，诞生了第一个研究所，第一个实验室，第一个研究生及其导师制度，第一个专业期刊，在这些众多第一的光环下，带领着德国的突飞猛进。

直到现如今，德国的教育仍然发达，有多达16所大学排在世界前百。看着德国教育体系的发展，让我不禁感叹，教育真的可以为国家逆天改命。
